Drawin Test Automation Framework

Prerequisite 
1) Ensure JRE is available in the system 
2) Ensure JAVA_HOME variable is SET in environment . To check , go to cmd prompt and type java -version , should display the version
 

To Run the Test : 

1)Extract the Compressed Test file 
2)Click the Runner bat file to execute the test 
3)Go to Report Folder to Check the Test results 
4)Go to TestData folder to modify any test data and Re-run the test
